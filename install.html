     Installation for WindowsXP  
 

  1. Install skystar2 driver
  2. Install software
  3. Configure the software
  4. Setup IP address
  5. Install GlobaX client
  6. Setup HTTP proxy
  7. Run Server4PC and GlobaX service
  8. Create VPN connection
  9. Install DVB Software

Parameters you need to connect

  • login
  • password
  • IP
  • Input speed
  • Output speed
  • PID
  • GlobaX server IP
  • GlobaX server Port
  • VPN server IP
 
1. Install skystar2 driver
 
After you plug in your skystar2 card the system should open Found New Hardware Wizard to setup the driver for it.
If skystar2 adapter is not installed yet then run Add New Hardware wizard by hand.
Menu Start / Settings / Control Panel / Add Hardware
 
You need to have driver installation package for your skystar2 card.
 
You need to install TechniSat DVB-PC TV Star PCI or B2C2 Broadband Receiver PCI Adapter.
 
Click Continue Anyway if there would be warning during driver installation.
 
You don't need to reboot the system after device installation.

2. Install software
   
Run Install\Setup.exe from SkyBroadcast Installer folder.
Setup the software step by step.
You don't need to reboot the system after software installation.

3. Configure the software
 
Menu Start / Programs / Broadband4PC / Setup4PC
 
Delete any existed or default satellite settings.
 
Add your settings for Skybroadcast connection.

Enter paramters:
  • Satelitte Name: skybroadcast-w6
  • LOF 1: 9750
  • Switch: 11700
  • LOF 2: 10600
Enter Transponder Settings in Transponder Management:
  • Transponder: 11595
  • Symbol Rate: 26662
  • FEC: Auto
  • Polary: Horizontal/Left (High)
Click Tune

Provide correct PID in Data Services
   Uncheck Hexadecimal checkbox
   Insert PID value (one of connection paramters)
   Check back Hexadecimal checkbox
4. Setup IP address
 
Setup specified IP address and netmask 255.255.255.0 for skystar2 network adapter.

5. Install GlobaX client
 
Use configurator3.exe tool from GlobaX client package to create globax.conf configuration file.

6. Setup HTTP proxy
 
Setup HTTP proxy for your browser as localhost:3128.

7. Run Server4PC and GlobaX service
   
Menu Start / Programs / Broadband4PC / Server4PC
Menu Start / Run Command: net start globax

8. Create VPN connection
 
Create new VPN network connection. Use VPN server IP from paramters you have.
 
Enter your login and password. Also setup in Properties CHAP authorization and disable encryption.

9. Install DVB Software
 
Now you can install DVB software.
 
Just use special installer.
 
Choose your language.
 
You don't need to reboot the system after installation.
